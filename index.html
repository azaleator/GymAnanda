Program Latihan Gym yang Disarankan

Senin Chest & Triceps
- Bench Press: 4 set x 12 rep
- Incline Dumbbell Press: 3 set x 12 rep
- Triceps Pushdown: 3 set x 15 rep
Selasa Back & Biceps
- Lat Pulldown: 4 set x 12 rep
- Barbell Row: 3 set x 12 rep
- Biceps Curl: 3 set x 15 rep

- Squat: 4 set x 12 rep
- Leg Press: 3 set x 12 rep
- Shoulder Press: 3 set x 12 rep
Jumat Full Body
- Deadlift: 3 set x 10 rep
- Push-ups: 3 set x 15 rep
- Pull-ups: 3 set x 8-12 rep
